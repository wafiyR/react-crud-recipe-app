# react-crud-recipe-app
CRUD (Create, Read, Update, Delete) Recipe App with React. Project was developed together with DSC MMU during JavaScript Week 2021.


Front-End:
React, HTML, CSS

Back-End:
Node.js, Express.js

Database:
PostgreSQL - Online Database Hosting with ElephantSQL
